# KPIs Library

This is a frontend architecture exercise.

## Getting Started

Install dependencies
```bash
npm install

# or

yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
